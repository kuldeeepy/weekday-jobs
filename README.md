## Run Locally

Clone the project

```bash
  git clone https://github.com/kuldeeepy/weekday-jobs.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the application

```bash
  npm run dev
```
